# Tecnicas_Programacion_Ej# Ejercicios de Técnicas de Programación

Este repositorio contiene ejemplos prácticos de las cuatro técnicas fundamentales de la Programación Orientada a Objetos (POO), desarrollados para la clase de Técnicas de Programación.

## Contenido del Repositorio

* **01_Herencia**: Ejemplo de cómo una clase hija (`Auto`) hereda atributos y métodos de una clase padre (`Vehiculo`).
* **02_Polimorfismo**: Demostración de cómo un mismo método (`hacerSonido`) puede comportarse de manera diferente en distintas clases.
* **03_Encapsulamiento**: Uso de modificadores de acceso `private` y métodos `get/set` para proteger la integridad de los datos.
* **04_Abstraccion**: Implementación de clases abstractas para definir estructuras base sin necesidad de instanciarlas.
